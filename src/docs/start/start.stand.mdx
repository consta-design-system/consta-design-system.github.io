# Быстрый старт

Добро пожаловать в Consta - дизайн-систему, которая поможет вам создавать красивые и функциональные интерфейсы для ваших веб-приложений. В этом разделе вы узнаете, как начать работать с Consta и создать свой первый проект.

## Импорт компонентов

1. Импортируйте в корневой файл библиотеку React, [тему](https://consta.design/libs/uikit/components-theme-stable/dev) и необходимые компоненты:

```jsx
import React from 'react';
import { Theme, $пресет } from '@consta/uikit/Theme';
import { $компонент } from '@consta/uikit/$компонент';
```

Где:

- $пресет — это набор токенов, которые определяют внешний вид компонентов. Вы можете импортировать несколько [пресетов](##LIBS.LIB.STAND.TAB/lib:uikit/stand:components-theme-stable/tab:dev/hash:пресеты) и использовать их в различных компонентах.
- $компонент — это название импортируемого компонента.

2. Вложите компоненты в тему:

```jsx
const App = () => (
  <Theme preset={$пресет}>
    <$компонент />
  </Theme>
);
```

### Пример

```jsx
import React from 'react';
import { Theme, presetGpnDefault } from '@consta/uikit/Theme';
import { Button } from '@consta/uikit/Button';

const App = () => (
  <Theme preset={presetGpnDefault}>
    <Button label="Кнопка" />
  </Theme>
);
```

## Настройка темы

Consta предоставляет несколько готовых тем:

- **Светлая тема** (GpnDefault): Традиционная светлая тема со светлым фоном и темным текстом. Хорошо подходит для большинства проектов и обеспечивает хорошую читаемость.

- **Темная тема** (GpnDark): Тема с темным фоном и светлым текстом. Темные темы обычно имеют привлекательный внешний вид и могут быть более приятными для работы в темных условиях освещения.

- **Акцентная тема** (GpnDisplay): Тема, которая объединяет как светлые, так и темные элементы в одном интерфейсе. Это может быть полезно для создания контрастных и выразительных дизайнов. Такая тема подойдет для видеостен, экранов с мониторингом, а также блоков, которые нужно выделить.

Вы можете создавать и подключать собственные темы с помощью [Конструктора тем](##THEME_CONSTRUCTOR).
