Цветовая палитра в дизайн-системе Consta — это набор цветов для создания визуального стиля и согласованного внешнего вида дизайна. Она включает в себя базовые и составные цвета. Любая цветовая палитра поддерживает светлую и темную тему.

Цель цветовой палитры — наследовать созданный набор цветов во все компоненты дизайн-системы.

# Базовые цвета

Используются для расчета составных цветов. Изменяйте базовые цвета, чтобы создать собственную цветовую палитру.

| Цвет                  | Описание                                                                                                    |
| --------------------- | ----------------------------------------------------------------------------------------------------------- |
| $color-base-base      | Базовый цвет содержимого, от которого выстраиваются цвета текста, иконок и элементов управления (контролов) |
| $color-base-essential | Базовый цвет поверхностей                                                                                   |
| $color-base-project   | Проектный цвет, от которого выстраивают акцентные состояния                                                 |
| $color-base-phantom   | Тонирующий цвет, от которого выстраиваются бордеры, оверлей под модальными окнами, скролл                   |
| $color-base-path      | Цвет ссылок и контролов, от которого выстраиваются все их состояния                                         |
| $color-base-system    | Цвет системный                                                                                              |
| $color-base-success   | Успех                                                                                                       |
| $color-base-alert     | Цвет ошибки                                                                                                 |
| $color-base-warning   | Цвет предупреждения                                                                                         |
| $color-base-caution   | Цвет осторожности                                                                                           |
| $color-base-normal    | Цвет нейтральный                                                                                            |
| $color-base-critical  | Цвет критичный                                                                                              |

# Составные цвета

Все составные цвета рассчитываются из базовых цветов путем изменения значений HSLA.

Доступно три группы составных цветов:

- Фоновые цвета.
- Цвета в типографике.
- Цвета контролов

Названия цветов в группах определяются семантически. Например, группа **Фоновые цвета** имеет название `--color-bg`, а цвет `--color-bg-tone` в этой группе отвечает за цвет затемняющей подложки под всплывающими окнами.

# Как настроить палитру

По умолчанию в дизайн-системе Consta доступно три цветовые схемы: светлая (Default), тёмная (Dark) и акцентная (Display).
